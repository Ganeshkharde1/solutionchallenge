# Food Donation System 
## Extra Food Donation system for event managers and NGO's


## To Run This File

### Clone the Repository on Local Machine


### Python SDK Version
```bash
Python 3.10 or higher version
```



### Install the Libraries
```bash
- pyrebase
- flask
- pickle
- numpy
- Tensorflow
- sklearn
  ```

### Open CMD in Current Directory

```bash
python app.py
```
